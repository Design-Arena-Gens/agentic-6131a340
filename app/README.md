## Hi Landing Page

A minimal, mood-aware greeting page built with the Next.js App Router. The hero section offers a friendly welcome, while the highlight cards outline the simple ways to kick off a conversation.

### Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the site locally.

### Available Scripts

- `npm run dev` – start the development server
- `npm run build` – create a production build
- `npm run start` – serve the production build
- `npm run lint` – run lint checks with ESLint

### Tech Stack

- Next.js (App Router, TypeScript)
- CSS Modules for scoped styling
- Geist fonts via `next/font`
